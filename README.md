## Architecture des dossiers
- Assets regroupe les fichier pour le front-end : img, js et css
- Functions pour les fonction backend
- controleur pour les routes qui recoivent et renvoie les donnée vers le front
- Les fichiers front en .html sont dans le dossier racine
